## How to use the App

Basic command:
```bash
npx my-custom-csv-parser
```
Console arguments:
1) Generates file of 14GB
```bash
 --generateFile <file>
```
2) Specify input filename path *required (if 1 was not provided)
```bash
--sourceFile <file>
```
3) Output filename path
```bash
--resultFile <file>
```
4) Separator char for .csv file ( "," by default)
```bash
--separator <char>
```
